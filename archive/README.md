# Archive (임시 글 보관 폴더)

이 폴더는 임시 글/메모를 보관하는 용도입니다.

- `archive/`는 `_config.yml`의 `exclude`에 등록되어 있어 빌드/배포 대상에서 제외됩니다.
- 여기 넣은 Markdown 파일은 사이트에 게시되지 않습니다.
- 카테고리처럼 하위 폴더로도 보관할 수 있습니다. 예: `archive/Examples/`
- 게시글 작성 규칙은 `POSTING_RULES.md`를 참고하세요.

예시:
- `archive/draft-idea.md`
- `archive/tmp-note-2026-03.md`
- `archive/Examples/post-01.md`
