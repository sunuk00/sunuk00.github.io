# 문자열 내장 함수

## input()
{% highlight python %}
name = input("이름이 머에요: ") # input()의 결과는 문자열
print(name)

a = int(input("a: "))
b = input("b: ")
print(a + float(b))

'''
실행결과

이름이 머에요: 휴버트 블레인 볼페슐레겔슈타인하우젠베르거도르프
휴버트 블레인 볼페슐레겔슈타인하우젠베르거도르프
a: 10
b: 20.5
30.5
'''
{% endhighlight %}

## sorted()
{% highlight python %}
words = ['candy', 'suger', 'honey', 'apple']
print(sorted(words))
print(sorted(words, reverse=True)) # 반대로 정렬
print(sorted(words, key=lambda x: x[4])) # 4번째 원소를 기준으로 정렬

'''
실행결과

['apple', 'candy', 'honey', 'suger']
['suger', 'honey', 'candy', 'apple']
['apple', 'suger', 'candy', 'honey']
'''
{% endhighlight %}

## format()
{% highlight python %}
a = 100
print("{} is my favorite number".format(a))
print(f"{a}kg is your weight")

print("{0} = {0}".format(100))
print("{0} != {1} = {1}".format(100, 200))

print("n1: {n1}, n2: {n2}".format(n1=5, n2=300))

'''
실행결과

100 is my favorite number
100kg is your weight
100 = 100
100 != 200 = 200
n1: 5, n2: 300
'''
{% endhighlight %}

`format()`보다 `f-string` 방식이 더 빠르다고 한다.

## split()
`split()` 함수는 문자열을 일정 기준으로 나누어 **list**에 담는다.
{% highlight python %}
s = "We live on the Earth, but someday we are going to go to Mars"
print(s.split(','))

'''
실행결과

['We live on the Earth', ' but someday we are going to go to Mars']
'''
{% endhighlight %}
