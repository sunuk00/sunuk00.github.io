다음의 코드는 에러가 발생한다.
`TypeError: 'str' object does not support item assignment`
{% highlight python %}
name = 'python'
name[0] = 'C'
name[:3] = 'CPP'
{% endhighlight %}


이유는 이렇다.   
파이썬에서 문자열(`str`)은 불변(immutable) 객체이다.
따라서 `name = 'python'`에서 `name`이 가리키는 문자열의 객체 일부 문자나 일부 구간을 직접 수정할 수 없다.

즉, 문자열은 immutable 객체이기 때문에 인덱스나 슬라이스를 이용해서 내부 값을 직접 변경할 수 없다.


그래서 아래와 같은 방법으로 문자열을 바꾼다.
{% highlight python %}
name = 'python'
name = 'C' + 'ython'
name = 'CPP' + name[3:]
{% endhighlight %}   
   

혹은 리스트로 바꿨다가 다시 문자열로
{% highlight python %}
name = 'python'
temp = list(name)
temp[0] = 'C'
name = ''.join(temp)
{% endhighlight %}
